DOM e Formulário

DOM

DOM, Data Object Model, é uma API(Application Programming Interface) que visa simplificara forma de acessar a estrutura de um documento HTML. Facilitando criar, modificar, adicionar e manipular elementos HTML.

Formulários

São elementos HTML que possibilitam a interação do usuário com a página web. Como exemplo temos caixas de texto, botões, caixas de seleção, dentr outros. As tags form, label, input, são elementos de formulários.

Exemplo

Construa uma interface para que sejam digitados dois números e somados.
